## Contribution

- Compatible tablettes et téléphones
- Testé sur Android 7 à 14
- Android 7.0 (API 24) et supérieur
**Compatibilité** :

- Taille APK : ~15MB
- Utilisation mémoire : ~50MB en moyenne
- Temps de compilation : < 100ms pour algorithmes simples
**Performance** :

- Limité aux algorithmes de taille moyenne
- Pas de débogueur interactif
- Pas d'auto-complétion
- Pas de coloration syntaxique avancée
**Limitations Connues** :

- Documentation exhaustive
- Support complet du langage Algo
- Compilation rapide
- Interface intuitive et moderne
**Points Forts** :

un IDE complet pour créer et exécuter des algorithmes sur Android.

## Format des Versions

- Exemples d'algorithmes inclus
- BUILD_GUIDE pour la compilation et le déploiement
- ARCHITECTURE détaillant la structure du code
- QUICKSTART pour démarrage rapide

### Documentation

- FileManager pour la persistance
- StateFlow pour la gestion d'état
- Coroutines pour les opérations asynchrones
- Jetpack Compose pour l'UI
- Architecture MVVM (Model-View-ViewModel)
### Architecture

- **Aléatoire** : aleatoire(), aleatoire_entre()
- **Conversion** : en_entier(), en_reel(), en_chaine(), en_caractere()
- **Tableaux** : taille(), ajouter(), inserer(), supprimer(), trier(), inverser()
- **Caractères** : ord(), chr()
- **Chaînes** : majuscule(), minuscule(), longueur(), sous_chaine(), concatener(), trouve()
- **Mathématiques** : racine(), abs(), puissance(), carre(), arrondi(), partie_entiere()
- **Saisie** : lire()
- **Affichage** : ecrire(), ecrireln()
### Fonctions Intégrées

- Expression conditionnelle : Si...alors...sinon (inline)
- Opérateurs de comparaison : =, <>, <, >, <=, >=
- Opérateurs logiques : et, ou, non (avec court-circuit)
- Opérateurs : arithmétiques (+, -, *, /, mod, div, ^)
- Tableaux unidimensionnels et multidimensionnels
- Fonctions et procédures avec paramètres et passage par référence
- Boucles : pour...de...à...pas, tantque, repeter...jusqua
- Instructions conditionnelles : si...alors...sinon, sinonsi, selon...cas
- Types de données : entier, réel, chaîne, caractère, booléen, tableau
