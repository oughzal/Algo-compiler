## 📖 Exemples de code avec coloration

Tous les exemples de code sont automatiquement colorés :

```algo
algorithme ExempleBoucle

variables
    i : entier
    somme : entier

debut
    somme = 0
    
    // Boucle avec pas
    pour i de 0 à 100 pas 10 faire
        somme = somme + i
        ecrireln("i = ", i, ", somme = ", somme)
    finpour
    
    ecrireln("Somme finale : ", somme)
fin
```

Le highlighter reconnaît :
- 🔴 Mots-clés en rouge
- 🔵 Types en bleu
- 🟢 Chaînes en vert
- 🟣 Fonctions intégrées en violet
- ⚫ Commentaires en gris italique

## 🎨 Personnalisation

### Modifier les couleurs

Éditez `assets/css/style.css` :

```css
:root {
    /* Couleurs principales */
    --primary-color: #0d6efd;
    
    /* Coloration syntaxique Algo */
    --algo-keyword: #d73a49;
    --algo-type: #005cc5;
    --algo-string: #22863a;
    --algo-number: #005cc5;
    --algo-comment: #6a737d;
    --algo-operator: #d73a49;
    --algo-builtin: #6f42c1;
}
```

### Ajouter un nouveau chapitre

1. Créez un fichier `chapitres/XX-nouveau-chapitre.html`
2. Copiez la structure d'un chapitre existant
3. Ajoutez le lien dans `index.html` et les menus de navigation
4. Utilisez `<code class="algo">...</code>` pour la coloration
