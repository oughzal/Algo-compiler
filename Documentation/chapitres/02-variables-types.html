🔢 Chapitre 2 : Variables et Types de Données

Qu'est-ce qu'une variable ?

Une variable est un espace en mémoire qui permet de stocker une valeur. Imaginez-la comme une boîte avec une étiquette (son nom) dans laquelle vous pouvez placer des données.

Analogie

Une variable est comme un casier dans un vestiaire :

  • Le numéro du casier = le nom de la variable
  • Ce qu'il contient = la valeur de la variable
  • Le type de contenu accepté = le type de la variable

Déclaration de variables

En Algo, les variables doivent être déclarées dans la section variables avant de pouvoir être utilisées.

Syntaxe de base

Déclaration simple
variables
    nomVariable : type

Déclaration multiple

Plusieurs variables du même type
variables
    x, y, z : entier
    nom, prenom : chaine

Les types de données

Le type d'une variable détermine quel genre de données elle peut contenir et quelles opérations on peut effectuer dessus.

1️⃣ Type entier

Stocke des nombres entiers (sans virgule) : ..., -2, -1, 0, 1, 2, ...

Exemple avec des entiers
algorithme ExempleEntiers

variables
    age : entier
    temperature : entier
    annee : entier

debut
    age = 25
    temperature = -5
    annee = 2026

    ecrireln("Âge : ", age)
    ecrireln("Température : ", temperature, "°C")
    ecrireln("Année : ", annee)
fin

2️⃣ Type reel (ou réel)

Stocke des nombres décimaux (avec virgule) : 3.14, -0.5, 2.0, ...

Exemple avec des réels
algorithme ExempleReels

variables
    prix : reel
    moyenne : reel
    pi : reel

debut
    prix = 19.99
    moyenne = 15.75
    pi = 3.14159

    ecrireln("Prix : ", prix, " €")
    ecrireln("Moyenne : ", moyenne, "/20")
    ecrireln("Pi ≈ ", pi)
fin
Important

En Algo, on utilise le point (.) comme séparateur décimal, pas la virgule (,).

3️⃣ Type chaine (ou chaîne)

Stocke du texte (une suite de caractères) : "Bonjour", "Alice", "123 rue de la Paix", ...

Exemple avec des chaînes
algorithme ExempleChaines

variables
    nom : chaine
    ville : chaine
    message : chaine

debut
    nom = "Alice Dupont"
    ville = "Paris"
    message = "Bonjour le monde !"

    ecrireln("Nom : ", nom)
    ecrireln("Ville : ", ville)
    ecrireln(message)
fin
À savoir

Les chaînes de caractères doivent être entourées de guillemets doubles (").

4️⃣ Type caractere (ou caractère)

Stocke un seul caractère : 'A', 'z', '5', '@', ...

Exemple avec des caractères
algorithme ExempleCaracteres

variables
    initiale : caractere
    symbole : caractere
    lettre : caractere

debut
    initiale = 'A'
    symbole = '@'
    lettre = 'z'

    ecrireln("Initiale : ", initiale)
    ecrireln("Symbole : ", symbole)
    ecrireln("Lettre : ", lettre)
fin
À savoir

Les caractères doivent être entourés d'apostrophes simples (').

5️⃣ Type booleen (ou booléen)

Stocke une valeur logique : vrai ou faux.

Exemple avec des booléens
algorithme ExempleBooleans

variables
    estMajeur : booleen
    aReussi : booleen
    estPair : booleen

debut
    estMajeur = vrai
    aReussi = faux
    estPair = (10 % 2 == 0)  // Résultat d'une comparaison

    ecrireln("Est majeur ? ", estMajeur)
    ecrireln("A réussi ? ", aReussi)
    ecrireln("10 est pair ? ", estPair)
fin

Récapitulatif des types

Type Description Exemples de valeurs Notation
entier Nombre entier -5, 0, 42, 1000 Sans guillemets
reel Nombre décimal 3.14, -0.5, 2.0 Avec point décimal
chaine Texte "Bonjour", "Alice", "" Entre guillemets doubles "..."
caractere Un seul caractère 'A', 'z', '5', '@' Entre apostrophes '...'
booleen Valeur logique vrai, faux Mots-clés vrai/faux

Affectation de valeurs

Pour donner une valeur à une variable, on utilise le symbole =.

Syntaxe d'affectation
nomVariable = valeur

Exemples d'affectation

Différents types d'affectation
algorithme Affectations

variables
    x : entier
    prix : reel
    nom : chaine
    initial : caractere
    actif : booleen

debut
    // Affectation de valeurs littérales
    x = 10
    prix = 29.99
    nom = "Alice"
    initial = 'A'
    actif = vrai

    // Affichage
    ecrireln("x = ", x)
    ecrireln("prix = ", prix)
    ecrireln("nom = ", nom)
    ecrireln("initial = ", initial)
    ecrireln("actif = ", actif)

    // Réaffectation (changer la valeur)
    x = 20
    prix = prix + 5.00
    nom = "Bob"

    ecrireln("\nAprès modification :")
    ecrireln("x = ", x)
    ecrireln("prix = ", prix)
    ecrireln("nom = ", nom)
fin

Initialisation lors de la déclaration

Il est possible d'initialiser une variable directement lors de sa déclaration :

Déclaration avec initialisation
algorithme InitialisationDirecte

variables
    age : entier = 25
    pi : reel = 3.14159
    nom : chaine = "Alice"
    actif : booleen = vrai

debut
    ecrireln("Âge : ", age)
    ecrireln("Pi : ", pi)
    ecrireln("Nom : ", nom)
    ecrireln("Actif : ", actif)
fin

Conversions de types

Parfois, il est nécessaire de convertir une valeur d'un type à un autre. Algo fournit des fonctions de conversion :

Fonction Description Exemple
versEntier() Convertit en entier versEntier(3.7) → 3
versReel() Convertit en réel versReel(5) → 5.0
versChaine() Convertit en chaîne versChaine(42) → "42"
Exemple de conversions
algorithme Conversions

variables
    x : entier
    y : reel
    texte : chaine

debut
    // Conversion réel → entier (troncature)
    y = 3.7
    x = versEntier(y)
    ecrireln("3.7 en entier : ", x)  // Affiche 3

    // Conversion entier → réel
    x = 5
    y = versReel(x)
    ecrireln("5 en réel : ", y)  // Affiche 5.0

    // Conversion nombre → chaîne
    x = 42
    texte = versChaine(x)
    ecrireln("42 en chaîne : ", texte)  // Affiche "42"
fin

Règles de nommage des variables

Conventions importantes
  • Le nom doit commencer par une lettre
  • Peut contenir des lettres, chiffres et underscores (_)
  • Pas d'espaces dans le nom
  • Évitez les mots-clés réservés (algorithme, debut, fin, si, etc.)
  • Utilisez des noms descriptifs et significatifs
✅ Valide ❌ Invalide Raison
age 2age Ne peut pas commencer par un chiffre
nom_complet nom complet Pas d'espaces autorisés
prix_TTC prix-TTC Le tiret n'est pas autorisé
ma_variable si Mot-clé réservé

Exemple complet

calculateur_imc.algo
algorithme CalculateurIMC

variables
    poids : reel
    taille : reel
    imc : reel
    nom : chaine

debut
    // Demander les informations
    ecrire("Entrez votre nom : ")
    lire(nom)

    ecrire("Entrez votre poids (en kg) : ")
    lire(poids)

    ecrire("Entrez votre taille (en m) : ")
    lire(taille)

    // Calculer l'IMC
    imc = poids / (taille ^ 2)

    // Afficher le résultat
    ecrireln()
    ecrireln("Bonjour ", nom, " !")
    ecrireln("Votre IMC est : ", imc)

    // Interprétation
    si imc < 18.5 alors
        ecrireln("Vous êtes en sous-poids.")
    sinonsi imc < 25 alors
        ecrireln("Vous avez un poids normal.")
    sinonsi imc < 30 alors
        ecrireln("Vous êtes en surpoids.")
    sinon
        ecrireln("Vous êtes en obésité.")
    finsi
fin
