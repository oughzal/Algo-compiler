Structure d'un algorithme

Structure complète
algorithme NomAlgorithme

constantes
    PI : reel = 3.14159
    MAX : entier = 100

variables
    x, y : entier
    nom : chaine

fonction maFonction(param : entier) : entier
variables
    resultat : entier
debut
    resultat = param * 2
    retourner resultat
fin

procedure maProcedure(param : entier)
debut
    ecrireln("Valeur : ", param)
fin

debut
    // Corps principal de l'algorithme
    x = 10
    ecrireln(x)
fin

variables

    x : entier

Opérateurs logiques

Opérateur Description Exemple
et / ET ET logique (court-circuit) x > 0 et x < 10
ou / OU OU logique (court-circuit) x < 0 ou x > 10
non / NON NON logique (négation) non actif
Court-circuit

Les opérateurs et et ou utilisent l'évaluation en court-circuit :
• A et B : B n'est pas évalué si A est faux
• A ou B : B n'est pas évalué si A est vrai

Entrées/Sorties

Fonction Description Exemple
ecrire(...) Affiche sans retour à la ligne ecrire("Bonjour")
ecrireln(...) Affiche avec retour à la ligne ecrireln("Bonjour")
lire(variable) Lit une valeur depuis l'entrée lire(x) 
// Affichage simple
ecrire("Hello ")
ecrire("World")  // Affiche: Hello World

// Avec retour à la ligne
ecrireln("Ligne 1")
ecrireln("Ligne 2")

// Affichage multiple
ecrireln("x = ", x, ", y = ", y)

// Échappements spéciaux
ecrireln("Ligne 1\nLigne 2")  // \n = retour ligne
ecrire("Col1\tCol2")          // \t = tabulation

// Lecture
lire(nom)          // Lit une chaîne
lire(age)          // Lit un entier
lire(prix)         // Lit un réel

Structures conditionnelles

Instruction si...alors...sinon

// Simple
si condition alors
    // instructions
finsi

// Avec sinon
si condition alors
    // instructions si vrai
sinon
    // instructions si faux
finsi

// Avec sinonsi
si condition1 alors
    // instructions
sinonsi condition2 alors
    // instructions
sinonsi condition3 alors
    // instructions
sinon
    // instructions par défaut
finsi

Expression conditionnelle

// Syntaxe
variable = si condition alors valeur1 sinon valeur2

// Exemple
max = si a > b alors a sinon b
statut = si age >= 18 alors "Majeur" sinon "Mineur"

Instruction selon (switch)

selon variable
    cas valeur1 : instruction1
    cas valeur2 : instruction2
    cas valeur3 : instruction3
    defaut : instructionParDefaut
finselon

Boucles

Boucle pour

// Ascendante (pas = 1 par défaut)
pour i de 1 à 10 faire
    ecrireln(i)
finpour

// Descendante (pas = -1 par défaut)
pour i de 10 à 1 faire
    ecrireln(i)
finpour

// Avec pas positif
pour i de 0 à 100 pas 10 faire
    ecrireln(i)  // 0, 10, 20, ..., 100
finpour

// Avec pas négatif
pour i de 100 à 0 pas -10 faire
    ecrireln(i)  // 100, 90, 80, ..., 0
finpour
Règles du pas
  • Le pas ne peut pas être 0
  • Si debut ≤ fin : pas par défaut = +1
  • Si debut > fin : pas par défaut = -1

Boucle tantque

tantque condition faire
    // instructions répétées tant que condition est vraie
fintantque

Boucle repeter...jusqua

repeter
    // instructions (exécutées au moins une fois)
jusqua condition  // répète jusqu'à ce que condition soit vraie

Tableaux

Déclaration

variables
    // Tableau 1D
    T : tableau[10] de entier
    noms : tableau[5] de chaine

    // Tableau 2D (matrice)
    M : tableau[3][3] de reel

    // Avec initialisation
    nombres : tableau[5] de entier = [1, 2, 3, 4, 5]

Accès aux éléments

// Lecture
valeur = T[0]      // Premier élément
valeur = T[9]      // Dernier élément (tableau de taille 10)

// Écriture
T[0] = 42
T[i] = valeur

// Matrices
M[0][0] = 1.0      // Première ligne, première colonne
valeur = M[i][j]
Indices

Les tableaux commencent à l'indice 0. Un tableau de taille N a des indices de 0 à N-1.

Passage par référence

// Avec ref, reference ou varref
procedure modifier(ref x : entier)
debut
    x = x + 1  // Modifie la variable originale
fin

// Utilisation
variables
    nombre : entier = 10
debut
    modifier(nombre)
    ecrireln(nombre)  // Affiche 11
