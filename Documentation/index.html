📚 Cours d'Algorithmique avec Algo Compiler

Bienvenue dans le cours complet d'algorithmique utilisant le langage Algo !

À propos de ce cours

Ce cours vous apprendra l'algorithmique de manière progressive, depuis les concepts de base jusqu'aux algorithmes avancés. Chaque chapitre contient des explications théoriques, la syntaxe Algo correspondante, et de nombreux exemples pratiques.

🎯 Objectifs du cours

  • Comprendre les concepts fondamentaux de l'algorithmique
  • Maîtriser la syntaxe du langage Algo
  • Développer une pensée logique et structurée
  • Résoudre des problèmes avec des algorithmes efficaces
  • Se préparer à la programmation dans d'autres langages

📋 Structure du cours

1️⃣ Introduction

Découvrez ce qu'est l'algorithmique, pourquoi elle est importante, et comment installer et utiliser Algo Compiler.

  • Qu'est-ce que l'algorithmique ?

2️⃣ Les Bases

Apprenez les éléments fondamentaux : variables, types de données, opérateurs, et interactions avec l'utilisateur.

  • Variables et Types de données
  • Opérateurs arithmétiques et logiques
  • Entrées et sorties (lire/écrire)

3️⃣ Structures de Contrôle

Maîtrisez les structures qui permettent de contrôler le flux d'exécution de vos algorithmes.

  • Les conditions (Si...Alors...Sinon)
  • Les boucles (Pour, TantQue, Répéter)
  • L'instruction Selon (Switch)

4️⃣ Structures de Données

Apprenez à organiser et manipuler des collections de données.

  • Tableaux (arrays)
  • Matrices (tableaux 2D)
  • Chaînes de caractères

5️⃣ Fonctions et Procédures

Structurez votre code avec des sous-programmes réutilisables.

  • Créer des fonctions et procédures
  • Utiliser les fonctions intégrées

6️⃣ Concepts Avancés

Explorez des concepts plus avancés pour résoudre des problèmes complexes.

  • La récursivité
  • Passage par référence
  • Algorithmes classiques (tri, recherche...)

7️⃣ Pratique

Mettez en pratique vos connaissances avec des exemples complets et des exercices.

  • Exemples complets commentés
  • Exercices avec solutions
  • Référence complète du langage

💾 Installation d'Algo Compiler

Prérequis
  • Java 21 ou supérieur (Télécharger Java)
  • Visual Studio Code (Télécharger VS Code)

Étapes d'installation

  1. Télécharger le compilateur
    📥 Télécharger algo-compiler-1.6.0.jar
  2. Installer l'extension VS Code
    🧩 Installer l'extension
  3. Configurer l'extension
    Dans VS Code, ouvrez les paramètres (Ctrl+,) et cherchez "Algo Compiler". Définissez le chemin vers le fichier JAR téléchargé.
  4. Créer votre premier algorithme
    Créez un fichier avec l'extension .algo et commencez à coder !

Premier programme

bonjour.algo
algorithme Bonjour

variables
    nom : chaine

debut
    ecrire("Entrez votre nom : ")
    lire(nom)
    ecrireln("Bonjour ", nom, " !")
fin
