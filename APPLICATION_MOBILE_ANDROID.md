## 🚀 Démarrage Rapide

### 1. Ouvrir dans Android Studio

```bash
cd algo-compiler-android
```

Puis ouvrir dans Android Studio : File → Open → Sélectionner le dossier

### 2. Synchroniser Gradle

File → Sync Project with Gradle Files

### 3. Exécuter

1. Lancer un émulateur ou connecter un appareil
2. Run → Run 'app' (Shift+F10)

### 4. Builder l'APK

```bash
# Windows
.\gradlew assembleDebug

# Linux/Mac
./gradlew assembleDebug
```

## 🎯 Fonctionnalités

### Écrans
